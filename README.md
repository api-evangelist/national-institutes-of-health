# National Institutes of HealthThis is a repo for managing machine readable artifacts for the [National Institutes of Health APIs](https://federation.nih.gov). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [National Institutes of Health APIs](https://federation.nih.gov).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://federation.nih.gov)- [Developer](https://federation.nih.gov)The [APIs.json](https://github.com/api-evangelist/national-institutes-of-health/blob/master/apis.json) index provides links to all the relevant links for the National Institutes of Health APIs, including the OpenAPI and Postman if they are present.
